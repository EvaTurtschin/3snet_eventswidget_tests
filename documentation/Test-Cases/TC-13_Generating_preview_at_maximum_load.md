## **TC-13 Генерация превью при максимальной нагрузке < 2сек**

***Priority***: P1
***Automation***: YES

***Steps***:

1. Выбрать Тематику = Выбрать все

2. Выбрать страны = Выбрать все

3. Ввести максимальное значение ширины 1020

4. Ввести максимальное значение высоты 720

5. Активировать radio button На всю ширину контейнера

6. Активировать radio button На всю высоту блока

7. Выбрать любую цветовую тему (не дефолт turquoise)

8. Нажать кнопку "Скопировать код"

9. Сгенерировать превью

***Expected result***:

- Измеренное время генерации превью не превышает допустимого (для примера возьмем 2сек)
