## **TC-06 Выбор цветовой темы**

***Priority***: P1
***Automation***: Yes

***Steps***:

1. Выбрать цветовую тему (не дефолт- turquoise)

2. Нажать кнопку "Сгенерировать превью"

***Expected result***:

- В src обновлён параметр theme и он соответствует выбору

- превью перезагружается с новой темой
