## **TC-12 Последняя выбранная цветовая тема перезаписывает предыдущие выборы**

***Priority***: P1
***Automation***: YES

***Steps***:

1. Выбрать первую цветовую тему (не дефолт, например purple)

2. Выбрать вторую цветовую тему (например blue)

3. Выбрать третью цветовую тему (turquoise)

4. Выбрать четвертую цветовую тему (например green)

5. Сгенерировать превью

***Expected result***:

- В src обновлён параметр theme на последнюю выбранную

- превью перезагружается с последней выбранной цветовой темой
